### 🌟 프로젝트 소개

코르크차지는 외식과 콜키지 문화가 일상화되는 시대에 사용자가 더 나은 선택을 할 수 있도록 공간과 문화를 '큐레이션'하는 서비스입니다. 변화하는 주류 문화 속에서 콜키지 시장의 정보 부족, 인식 부족, 비즈니스 부족을 해결하여 새로운 주류 문화를 선도하고자 합니다.


👥 **내 역할:**

- **UI 개발**: 콜키지맵, 해주세요, 바텀시트 컴포넌트 구현
- **API 연동**: **네이버 맵 api 연동**, 매장 저장 api 연동, 지도부분 자체 api 연동
- **그 외 작업**: **웹 도메인 vercel 배포, github action 를 통한 ci/cd 구축**, 개발 컨벤션을 포함한 문서 작업

### ✨ 주요 기능

- **콜키지 맵 (Corkage Map)** (**주 담당 파트**)
    - 네이버 지도를 활용하여 지도 위에 콜키지 정보를 핀 형태로 표시합니다.
    - 사용자의 위치를 기반으로 주변 매장 정보를 제공합니다.
- **가게 상세 정보 (Shop Detail Info)** 
    - 네이버 지도 데이터를 활용해 매장의 영업시간, 주소 등 모든 정보를 한 번에 확인할 수 있습니다.
    - 대표 메뉴와 어울리는 페어링을 AI가 추천해 줍니다.
- **해주세요 (Please Do It Service)** (**주 담당 파트**)
    - 매장에 직접 문의하지 않아도, 코르크차지 팀이 대신 정보를 확인하고 등록하는 콜키지 전문 심부름 서비스입니다.
- **콜키지 팁 (Corkage Tips)**
    - 콜키지 방법, 매너, 페어링 등 다양한 정보를 아티클 형태로 제공하여 콜키지 문화의 진입장벽을 낮춥니다.
- **콜키지 스코어 (Corkage Score)**
    - 콜키지 경험에 대한 리뷰를 작성하고 공유하는 시스템입니다.
- **지금 핫한 매장 (Hot Shops Now)**
    - 콜키지를 서비스하지 않는 가게의 '저장' 수가 기준치(100개)를 넘으면, 코르크차지 팀이 직접 영업하여 콜키지 도입을 제안합니다.
- **콜키지 추가하기 (Add Corkage)** (**주 담당 파트**)
    - 영업사원이 현장에서 빠르게 콜키지 정보를 입력할 수 있는 기능입니다.
- **주류예약 (Reserve Bottle)**
    - 데일리샷과의 연계를 통해 방문하고자 하는 가게와 가까운 곳에서 합리적인 가격에 주류를 픽업 예약할 수 있습니다.
